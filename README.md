## Setup

Use `nvm` to install the correct version of node:
```shell script
nvm install
```

Copy `.env.example` to `.env`.
```shell script
cp .env.example .env
```

Install packages.
```shell script
npm install
```

## Usage

Run the app.
```shell script
npm run dev
```

Run the linter.
```shell script
npm run lint
```

Run the tests.
```shell script
npm test
```
